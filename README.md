# claude-code-agent-prompt-hud

Lightweight Agent HUD Prompt for Claude Code / AI CLI.

## Features

- 中文 / English HUD
- Session Time
- Token Usage
- Cost Estimation
- Git Changes
- Context Files
- CPU / Memory
- Request Counter
- Branch / Directory

## Install

bash install.sh

## Switch Language

export HUD_LANG=en
source ~/.zshrc
